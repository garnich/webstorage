WebStorage demo

LocalStorage
  • Locally Stored Data
  • Stores data with no expiration date
  • Data is never transferred to the server
  • Storage limit is at least 5MB
SessionStorage
  • Locally Stored Data
  • The sessionStorage object stores data only for a session
  • Data is never transferred to the server
  • Storage limit is at least 5MB
Cookies
  • Stores data that has to be sent back to the server with subsequent requests.
  • Primarily for server-side reading (can be read on client)
  • Size less than 4KB
IndexedDB
  • Key/value storage: value can be (almost) anything
  • Is intended for offline apps
  • Storage limit is calculated as 50% of free disk space
